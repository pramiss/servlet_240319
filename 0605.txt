Servlet에서 DB연동
JDBC : Java DB Connectivity
과정 : WAS -> (JDBC) -> MySQL or Oracle
프로그램이 DB에 직접 접근하지 않고 JDBC를 통해 접근함
-> 이것도 Spring에서는 우리가 직접하지 않음


jar 확장자 : Java 라이브러리 묶음
connector-j 내부의 jar 파일을 
프로젝트>Java Resources>Libraries>Tomcat 내부에 추가하면 됨
방법 : 파일탐색기 > tomcat 폴더 > lib 에 jar파일 복붙하고 이클립스가서 f5하면 됨


commandline 또는 workbench -> MySQL : 이전까지 한 방법
WAS -> MySQL : 오늘할 방법
DB port: 3306, WAS port: 80


연결할 클래스가 필요함
com.test.common.MysqlService
(common 패키지: 모든곳에서 공통으로 사용한다는 의미)
(MysqlService 클래스: DB에 접속하기 위한 설정 클래스)
