Spring framework vs Spring boot

<Spring 책 추천>
Spring 3.0, 3.2 => 토비의 스프링 (입사 후 코드에 익숙해진 후 읽어보기)


<Request  ->  Response>
                @Controller + @ResponseBody : return String => HTML
                @Controller + @ResponseBody : return Map => JSON String
                @RestController = @Controller + @ResponseBody
                @RestController : return String => HTML
                @RestController :  return Map => JSON String
                @RestController :  return (내가 만든 객체) => JSON String

@ResponsBody + return String -> HttpMessageConverter(클래스) -> HTML
@ResponsBdoy + return 객체(Map, List, 사용자정의 객체)
                -> HttpMessageConverter(클래스) -> jackson lib -> JSON

HttpMessageConverter의 역할: 나중에 Spring Framework를 할때 도움됨
